# Design reference

Visual spec for the frontend, authored with **Claude Design** (Anthropic's
web-based design tool). This is a **reference only** — extract layout, spacing,
typography, and color values, then reimplement from scratch under the frontend
rules in [CLAUDE.md](../CLAUDE.md). Do not import, copy, or port its markup,
runtime, or dependencies (React, etc.) into the app.

- `IRC Client.dc.html` — the exported mockup.
- `support.js` — the runtime the mockup loads (`<script src="./support.js">`),
  generated by Claude Design. It is **not published**: it's gitignored (see
  `.gitignore`) because the bundled `dc-runtime` has no clear redistribution
  license, and it was never part of the product anyway — nothing in `design/`
  is built or embedded (the binary embeds only `web/dist/`). To render the
  mockup locally, keep/restore your own `design/support.js` from Claude Design.

by the repository's AGPL-3.0. The application code that ships (everything outside
